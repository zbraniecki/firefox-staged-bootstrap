A proof of concept of a stages bootstrap architecture for Firefox.

At the moment no UI is provided, so the only observables are in the console log.

The following code exist:

  • bootstrap.js - Bootstrap and BootstrapStage API
  • bootstrap-analysis.js - very trivial measurement dumping
  • main.js - POC of code actually scheduling tasks

Source: https://github.com/zbraniecki/firefox-staged-bootstrap
